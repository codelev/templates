# Templates
Sole Owner Limited Liability Company registration templates. Actual by May 2026.

The templates are in [Typst](https://github.com/typst/typst) format, which lets you create a complete set of company documents in minutes.
Live preview in IDEA is optional but useful, requires [Typst Renderer](https://plugins.jetbrains.com/plugin/31308-typst-renderer) plugin installed.

## Usage

1. Copy `style.typ` to `style.local.typ`.
2. Update `style.local.typ` with your details.
3. Install PDF converter: `sudo snap install typst`.
4. Generate PDF files e.g. `typst compile incorporation-act.typ`
